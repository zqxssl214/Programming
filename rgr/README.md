# Генератор лабиринта и поиск пути

Программа на C++ для генерации случайного лабиринта и поиска кратчайшего пути от старта до финиша.

## Структура проекта

- `main.cpp` — точка входа, обработка аргументов командной строки и конфигурационного файла.
- `maze.cpp` — реализация генерации лабиринта (бэктрекинг и Эллер) и поиска пути (BFS и A*).
- `maze.h` — заголовочный файл с описанием структуры Maze и прототипов функций.
- `logger.cpp` — реализация логирования в файл с уровнями ERROR/WARNING/INFO.
- `logger.h` — заголовочный файл логгера.
- `tests/tests.cpp` — юнит-тесты для проверки работоспособности.
- `CMakeLists.txt` — конфигурационный файл для сборки проекта.
- `config.cfg` — конфигурационный файл программы.

## Конфигурация приложения

Пример содержимого `config.cfg`:

- `WIDTH` / `HEIGHT`: размеры лабиринта (рекомендуются нечётные числа, минимум 5).
- `GENERATOR`: алгоритм генерации. 1 — рекурсивный бэктрекинг, 2 — алгоритм Эллера.
- `FINDER`: алгоритм поиска пути. 1 — BFS, 2 — A*.

## Аргументы командной строки

Параметры можно переопределить через командную строку:


Примеры:
```bash
./maze 21 21 1 1     
./maze 21 21 2 2     
./maze 15 15 1 2     
```

## Сборка

```bash
mkdir build && cd build
cmake ..
make
```
## Запуск
```bash
./maze 21 21 1 1
```

## Запуск тестов
``` bash
./tests
```
